# Font Size Guide for Academic Presentation

## Current Issues to Fix:
1. All text needs to be 50-100% LARGER
2. Background needs to be warm cream/beige (NOT dark slate)
3. Cards need white/cream backgrounds with subtle shadows

## New Font Sizes (Apply Everywhere):
- Headers (المحور): `text-lg` → should be visible from distance
- Main Titles: `text-4xl md:text-6xl` (huge!)
- Section titles: `text-2xl md:text-3xl`
- Body text: `text-xl` (very readable)
- Labels: `text-lg`
- Small text: `text-base` (minimum)
- Icons: `w-8 h-8` minimum

## New Color Scheme:
- Background: `bg-gradient-to-br from-amber-50 via-orange-50 to-amber-50`
- Cards: `bg-white` with `shadow-lg`
- Borders: `border-gray-200`
- Text Main: `text-gray-900`
- Text Secondary: `text-gray-600`
- Primary (Algerian Telecom Green): `text-emerald-600` / `bg-emerald-600`
- Accent (Orange): `text-orange-600` / `bg-orange-600`
